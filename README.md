> This project is a responsive, full-stack platform designed to centralize and streamline the management of student organizations and their members within a college ecosystem. It provides administrators and club coordinators with robust tools to manage events, track resources, and enhance campus engagement.

## Features
- **Role-Based Access Control (RBAC):** Implementation of 4 distinct dashboards tailored for Administrators, Coordinators, and Students.
- **Event Management & Registration:** Modules for creating events, handling registrations, and tracking attendee lists.
- **Donation & Finance Tracking:** Secure integration with Razorpay for tracking club donations.
- **Library Management System:** A dedicated module for managing club-specific resources or books.
- **Rich Media Integration:**
  - **Cloudinary:** For efficient image and asset storage/delivery.
  - **YouTube API:** To embed and manage club video content within the gallery.
- **Centralized Announcement System:** A global feed for broadcasting updates across the platform.
